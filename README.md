# JavaScriptmas – Day 2: Ugly Advent Calendar

This is my **Ugly Advent Calendar** created for **JavaScriptmas Day 2**.  
I customised the colors, layout, and box behavior to make it clearer, more readable, and still intentionally “ugly” in the spirit of the challenge.

---

## ✨ Main Features

- **24 interactive boxes**, numbered 1–24  
- Each box reveals a **random Christmas gift**  
- Boxes **open and close** with a click  
- Box state is saved using **localStorage**  
- **Quick reset** by pressing the `R` key  
- **Custom colors** (opened = green, closed = red)  
- Improved title font, spacing, and layout
